Project DBS Archives

This project involved the creation of a dedicated subsite within the DBS Archives, serving as a captivating showcase of the bank's rich history, replete with historical events and impressive milestones achieved over the years. The most demanding challenge we encountered lay in the implementation of a scrolling timeline effect. This feature was pivotal in providing visitors with an immersive journey through DBS's history, making it both engaging and informative.

A scrolling timeline showcasing past DBS CEOs as users scrolled down. The challenge was to seamlessly combine creative design with precise technical execution to create a visually captivating experience. This timeline served as the key feature of DBS's historical archive subsite, providing users with a dynamic and technically advanced journey through the bank's CEO history.

I developed a sorting and filtering system with pagination using only vanilla HTML, CSS, and JavaScript, without relying on external plugins. This implementation enhances the user experience by allowing seamless organization and access to content based on various criteria.

As part of this project's content migration task, I designed and implemented a custom PHP function akin to WordPress' WYSIWYG editor. This tool simplifies the process of populating articles and content, streamlining the migration effort and ensuring a user-friendly approach to content management. Saving me hours of time compared to doing it manually.
